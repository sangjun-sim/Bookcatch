# Bookcatch UI Source Snapshot

This repository is a UI source snapshot for Bookcatch.

It is not a buildable application repository. The snapshot is intended to show
selected Flutter UI structure, styling, localization text, widgets, and screen
composition without publishing the full app internals.

The public snapshot excludes tests, models, data, platform setup, sync logic,
repository layers, local database code, and runtime configuration files.

Included areas are limited to selected UI-facing files such as theme code,
shared widgets, localization files, and representative feature screens.
